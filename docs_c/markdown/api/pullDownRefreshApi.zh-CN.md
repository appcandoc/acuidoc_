### appcan.startPullDownRefresh(OBJECT)

页面开始下拉事件

参数 | 类型 | 必填 | 说明
---|---|---|---
success | Function | 否 | 开始下拉执行的回调

### appcan.onPullDownRefresh(OBJECT)

监听下拉中事件

参数 | 类型 | 必填 | 说明
---|---|---|---
success | Function | 否 | 监听下拉中回调

### appcan.stopPullDownRefresh(OBJECT)

停止下拉事件

参数 | 类型 | 必填 | 说明
---|---|---|---
success | Function | 否 |停止下拉回调
